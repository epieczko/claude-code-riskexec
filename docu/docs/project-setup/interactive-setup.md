# Interactive Project Setup

The `claude-code-templates` CLI provides an interactive way to set up and configure your projects for Claude Code. This is the recommended way to get started.

## Quick Start

To begin, navigate to your project directory and run the following command:

```bash
npx claude-code-templates@latest
```

From this interactive menu, you can choose to:
- **📊 Analytics Dashboard**: View your Claude Code sessions, usage patterns, and performance metrics.
- **🔍 Health Check**: Perform a comprehensive system validation and configuration verification.
- **⚙️ Project Setup**: Configure Claude Code for your project with framework-specific commands and hooks.

Choosing "Project Setup" will guide you through the process of setting up your project with the appropriate Claude Code templates and configurations. This includes:
- Auto-detection of your project's language and framework.
- Generation of `CLAUDE.md` with project-specific instructions.
- Configuration of development commands.
- Installation of Claude Code agents for specialized assistance.
